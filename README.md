Google code-in: REST and JSON: Make an executable program (html, java, c, c++ etc)
-------------
-------------

This is a Google code-in project. It is written in C++.

Dependency:
----------
-libcurl
-libjsoncpp
-C++11

First build it:

git clone https://github.com/DengYiping/RESTpp.git

cd RESTpp

cd RESTpp

make

Then run:

./RESTpp
